A simple Python Tkinter GUI app that displays random jokes offline using pyjokes. Click Get Joke and enjoy! Works with Python 3.x, no internet needed.

Features

Lightweight and offline

Easy-to-use GUI built with Tkinter

One-click joke generation

Cross-platform: Windows, Linux, macOS

Fully offline; no extra setup except installing pyjokes


Installation Steps: 

Clone the repository:

Step 1:git clone https://github.com/DeveloperSandesh/joke-generator.git
Step 2: cd joke-generator
Step 3: pip install -r requirements.txt
Step 4: python main.py
